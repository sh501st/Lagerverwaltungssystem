• Left sidebar allows changing the storage dimensions via the width and height sliders.
  • Left click (and hold) on tiles on the right-hand side to place shelves.
  • Middle click on an edge tile allows placing a storage entrance.
  • Right clicking (and holding) on tiles removes shelves or entrances.
  • Unreachable shelves are marked and need to be removed before the storage setup is accepted.
  • To switch to the live view, accept via "Save and view".
  • The Cancel button brings you back to the main menu.
